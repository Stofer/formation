Le Xhtml oblige que tout document Xhtml commence par un élément de type bloc (block). Les validateurs
ne manqueront pas de vous rappeler tout manquement à cette règle.
Ainsi, toute la page sera structurée ou divisée en éléments bloc qui distingueront des parties entières de
texte. Nous avons déjà rencontré deux éléments blocs dans notre étude: les titres et les paragraphes.
Les éléments de bloc créent une nouvelle ligne dans le flux de texte. Suivant le type de l'élément, le
navigateur peut alors insérer automatiquement un espace plus important pour séparer du contenu précédent.
Ainsi les navigateurs créent-ils, par exemple, pour la balise P, un espace visible pour que le changement
de paragraphe soit distingué du simple passage à la ligne.
Les éléments bloc se placent toujours, par défaut, l'un en dessous de l'autre.
